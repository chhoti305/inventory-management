# Inventory Management System

A robust **Java + MySQL** Inventory Management System built with Maven, following the **DAO Design Pattern**.

## ✨ Features
- Full CRUD operations (Create, Read, Update, Delete)
- Low stock alert system
- Category-wise stock value aggregation
- Sort items by total value (descending)
- Summary dashboard with key metrics
- Transaction tracking structure (IN/OUT)
- Input validation and exception handling

## 🛠 Tech Stack
- **Language**: Java 11
- **Build Tool**: Maven
- **Database**: MySQL 8
- **Architecture**: DAO + Service Layer
